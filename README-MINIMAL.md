# Cate Sandstrom - Website

Personal website for Traditional Irish Fiddler and musician.

## What's Here

- Music performances and videos
- Irish fiddle and classical violin lessons
- Performance calendar
- Workshop information

## Tech

- Static HTML/CSS/JS
- Hosted on GitHub Pages
- Custom domain: catesandstrom.com

## Local Development

1. Clone this repo
2. Open `index.html` in a browser
3. Edit files directly
4. Commit and push to deploy

## Structure

```
/css        - Stylesheets
/js         - JavaScript (mobile menu)
/images     - All photos and favicon
*.html      - Page files
```
