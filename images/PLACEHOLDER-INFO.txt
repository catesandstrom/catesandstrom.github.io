HOW TO ADD YOUR PHOTOS:
1. Resize photos to recommended dimensions (or close to them)
2. Name them exactly as listed above
3. Upload to the /images folder
4. Photos should be .jpg format for best web performance
5. Keep file sizes under 500KB each for fast loading

TIPS:
- Use high-quality photos but compress them for web
- Tools like TinyPNG or Squoosh can compress without quality loss
- Landscape photos work best for hero/carousel images
- Square or portrait photos work best for about/contact pages
